# 打卡应用 - 习惯养成的可视化工具

_2026-03-15_

今天介绍我的第三个项目：**打卡应用**。

<https://wangshuang12138-beep.github.io/habit-tracker/>

## 项目背景

想养成一个习惯，最难的是坚持。而坚持的前提是「看见」——看见自己的努力，看见连续的天数，看见时间的痕迹。

这个应用最初是为口语练习设计的，但也可以用来追踪任何周期性任务：跑步、阅读、练琴、写作...

## 核心功能

### 📅 灵活打卡
- 支持每日、每周、自定义周期
- 随时补打过去的记录，不会因为漏了一天就放弃

### 📝 多种记录方式
- 文字记录：练习内容、心得体会
- 图片打卡：截图、照片作为凭证

### 👤 多用户支持
可以创建多个账号，追踪不同的习惯项目。

### 📊 Apple 风格的数据可视化
- **圆环进度**：今日/本周完成度
- **趋势图表**：查看打卡频率变化
- **年度热力图**：一眼看出全年的坚持情况

## 技术架构

这是一个前后端分离的项目：

| 部分 | 技术 | 部署 |
|------|------|------|
| 前端 | Vue 3 + Vite | GitHub Pages |
| 后端 | Express.js | Vercel |
| 数据 | JSON 文件存储 | Vercel 服务器 |

选择 JSON 文件而非数据库，是因为这个项目面向个人使用，数据量小，部署简单。

## 设计细节

界面参考了 Apple 健康 App 的视觉风格：
- 圆角卡片
- 柔和阴影
- 鲜明的数据色彩
- 清晰的层级关系

目的是让「打卡」这件事本身就有成就感。
