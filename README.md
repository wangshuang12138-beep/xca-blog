# xca-blog

xca 的个人博客，使用 VitePress 构建。

## 在线访问

https://wangshuang12138-beep.github.io/xca-blog/

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build
```

## 目录结构

```
xca-blog/
├── .vitepress/         # VitePress 配置
│   ├── config.mjs      # 站点配置
│   └── theme/          # 自定义主题（可选）
├── posts/              # 博客文章
├── index.md            # 首页
├── about.md            # 关于页面
└── logo-xca-square.svg # 站点 logo
```

## 写作

文章放在 `posts/` 目录，使用 Markdown 格式。
