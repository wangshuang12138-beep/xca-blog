# 决策骰子 - 让 AI 帮你做选择

_2026-03-15_

今天介绍我的第二个开源项目：**决策骰子**。

<https://wangshuang12138-beep.github.io/decision-dice/>

## 为什么做这个

生活中总有纠结的时刻：
- 中午吃什么？
- 周末去哪玩？
- 要不要买这个？

小事纠结浪费时间，大事纠结消耗精力。我想做一个工具，既能快速给出选择，又能提供理性分析。

## 功能设计

### 🎯 随机决策
输入 2-8 个选项，一键随机选择。没有纠结，只有结果。

### 🤖 AI 辅助分析
接入 Kimi AI，在给出结果的同时分析每个选项的利弊。不是替你做决定，而是帮你理清思路。

### 📝 决策复盘
决策后可以对结果进行评价（👍/👎/🤷），积累自己的决策数据。时间久了，你会发现自己到底擅长在什么场景下做选择。

### 📊 数据统计
- 总决策数
- AI 辅助次数  
- 决策满意度趋势

## 技术实现

- **Vue 3 + Vite** - 响应式框架，开发体验好
- **Pinia** - 状态管理，处理决策历史和统计数据
- **Kimi API** - AI 分析功能，用户可自行配置 API Key
- **GitHub Pages** - 免费托管，自动部署

数据全部保存在浏览器本地，不涉及隐私问题。

## 使用场景

| 场景 | 用法 |
|------|------|
| 日常纠结 | 输入选项，随机选择 |
| 重要决策 | 开启 AI 分析，获得多角度建议 |
| 决策训练 | 坚持复盘，提升决策直觉 |
