# 博客搭建记录

_2026-03-15_

这是我的第一篇博客文章，记录一下搭建这个博客的过程。

## 为什么用 VitePress

之前有 Vue 项目经验，VitePress 是 Vue 团队出品，上手快，默认主题就很简洁，符合我的审美。

## 设计过程

### Logo

博客 logo 是 **xca**，来自我的名字"爽"（shuang）在方言里的发音 —— x 变成了 ca。

设计风格参考小米，极简的圆角方块 + 文字。

### 功能

保持精简：
- 首页展示
- 文章列表
- 关于页面
- 深色模式
- 搜索

## 部署

用 GitHub Actions 自动部署到 GitHub Pages，每次 push 到 main 分支就自动更新。
